# rock-paper-scissors-ml
Solution for the rock paper scissors task from freeCodeCamp

FreeCodeCamp reference: https://www.freecodecamp.org/learn/machine-learning-with-python/machine-learning-with-python-projects/rock-paper-scissors

Task: Achieve win rate >= 60% against four opponents using machine learning.

I had a solution previously on https://replit.com/@mark182182/rock-paper-scissors, but that got deleted due to the Replit structuring.
This previous solution solution was using Q-learning by pre-training a model that is playing 10 matches in 200 rounds in the following order:

kris
abbey
quincy
mrugesh
random
abbey
